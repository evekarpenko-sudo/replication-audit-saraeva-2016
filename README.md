# replication-audit-saraeva-2016
Replication audit of Saraeva (2016): transparency, reproducibility and open science assessment
## Описание проекта

В данном репозитории представлены материалы репликационного аудита статьи:
Сараева Н. М. «Эмпирическое изучение субъектности студентов: экопсихологический подход» (2016).
Работа выполнена в рамках дисциплины «Планирование эмпирического и экспериментального исследования».

## Содержимое репозитория

### report
Содержит полный текст репликационного отчёта.

### tables
Содержит таблицы и результаты оценки воспроизводимости.

### materials
Содержит исходную статью, выбранную для аудита.

### code
Содержит материалы анализа. Вычислительный код в рамках проекта не использовался.

## Основные выводы

- Концептуальная репликация исследования возможна.
- Полная статистическая репликация невозможна.
- Исходные данные отсутствуют.
- Алгоритмы обработки данных описаны недостаточно подробно.
- Уровень прозрачности исследования оценивается как низкий.
